Source Document Clickable

Enhance your workflow with the 'CR Source Document Clickable' module, enabling one-click navigation to source documents such as Delivery Orders, Purchase Orders, and Manufacturing Orders for increased convenience.
This module allows to click and navigate the origin document.Just install module and all source document on Delivery Order, Purchase Order, Manufacturing Order are clickable.
